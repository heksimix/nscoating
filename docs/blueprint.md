# **App Name**: OrderFlow

## Core Features:

- Add New Order: Functionality to add new orders with customer details, date, and item details.
- View Order List: Display a list of all orders.
- Edit Order: Functionality to edit existing orders.
- Search/Filter Orders: Allows searching and filtering orders by customer or date.
- Order Summary Generator: Generate summaries of each order, detailing key information such as total cost, items, customer, etc, as a tool to make decisions about order statuses
- Database Integration: Stores customer orders.

## Style Guidelines:

- Primary color: Teal (#26A69A) for a fresh, professional look.
- Background color: Light cyan (#E0F7FA) for a clean interface.
- Accent color: Lime green (#AEEA00) for highlighting important elements and actions.
- Body and headline font: 'Inter' sans-serif, suitable for headlines or body text, providing a modern and neutral look.
- Use minimalist icons to represent order status and actions.
- Clean, intuitive layout for easy navigation and order management.
- Subtle animations on data changes to reflect those in the UI, such as editing orders.
